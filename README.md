## download:

Remember to keep the same matching version **X**
- chromium version X
- chromedriver version X

## install:

```bash
$ python3 -m venv env/
$ source env/bin/activate
$ python3 -m pip install -r requirements.txt
```

## run this when finished:

```bash
$ deactivate
```

## run

```bash
$ python3 ifood_gift_card.py
```
